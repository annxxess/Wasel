## 📌 About WASEL

WASEL (واصل) is a smart intra-city delivery platform that connects customers,
delivery drivers, stores, restaurants, pharmacies, and businesses in one
intelligent ecosystem.

---

## 🛠️ Tech Stack

| Layer            | Technology                          |
|------------------|-------------------------------------|
| Mobile App       | Flutter                             |
| Web Frontend     | React.js + Tailwind CSS             |
| Backend          | Node.js + Express.js                |
| Database         | PostgreSQL                          |
| Auth             | JWT Authentication                  |
| Maps & GPS       | Google Maps API                     |
| Notifications    | Firebase Cloud Messaging            |
| Storage          | Cloudinary                          |
| Hosting          | Vercel / Railway / Render           |
